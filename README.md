# Парсер погоды за неделю

Скрипт собирает данные о погоде (температура, осадки) на неделю вперёд для заданного города с веб‑сайта и сохраняет их в CSV/Excel.

## Используемые библиотеки
- `requests` — загрузка веб‑страницы.
- `BeautifulSoup4` — парсинг HTML.
- `pandas` — обработка и анализ данных.
- `time`, `datetime` — вспомогательные модули.
